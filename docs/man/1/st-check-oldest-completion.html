    NAME

    st-check-oldest-completion - a Swift-tools command

    SYNOPSIS

    st-check-oldest-completion [OPTION]...

    DESCRIPTION

    st-check-oldest-completion check if a Swift cluster has swiftstore(s) with a last synchronization that is too old

    [-D|--max_days]

    Max days for oldest completion. Sum this with max_hours

    [-H|--max_hours]

    Max hours for oldest completion. Sum this with max_days
