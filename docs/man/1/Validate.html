    NAME

    Validate - simple validation tool for XML documents

    SYNOPSIS

    Validate dtdfile xmlfile

    DESCRIPTION

    Validate is a simple validation tool for XML documents, which validates the XML file using the DTD file and reports all the errors it can find.
