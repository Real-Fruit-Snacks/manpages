    NAME

    ctsimtext - A Text-Based Computed Tomography Simulator

    SYNOPSIS

    ctsimtext [processing-command processing-options]]

    DESCRIPTION

    ctsimtext is a text user interface to the various components of the CTSim package. The home for CTSim is http://www.ctsim.org.

    OPTIONS

    ctsimtext accepts the following options

    --version Display version
    --help Print a terse help message

    AUTHORS

    Kevin Rosenberg, M.D. <kevin@ctsim.org>

    HISTORY

    CTSim was begun in 1983 using MS-DOS and an EGA display adapter. In 1999 it was ported to GNU/Linux and later ported to Microsoft Windows.
