    NAME

    pgpainless-cli-help - Stateless OpenPGP Protocol

    SYNOPSIS

    pgpainless-cli help [-h] [--stacktrace] [COMMAND]

    DESCRIPTION

    When no COMMAND is given, the usage help for the main command is displayed. If a COMMAND is specified, the help for that command is shown.

    OPTIONS

    -h, --help

    Show usage help for the help command and exit.

    --stacktrace

    Print stacktrace

    ARGUMENTS

    [COMMAND]

    The COMMAND to display the usage help message for.
