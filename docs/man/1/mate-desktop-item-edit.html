    NAME

    mate-desktop-item-edit - A small tool to edit .desktop files.

    SYNOPSIS

    mate-desktop-item-edit [OPTIONS] [FILE]

    DESCRIPTION

    Distributed with mate-panel, mate-desktop-item-edit allows you to edit .desktop files.

    OPTIONS

    --create-new
    Create a new file in the given directory.
    --display=DISPLAY
    X display to use.
    -?, -h, --help
    Print standard command line options.
    --help-all
    Print all command line options.

    This program also accepts the standard GTK options.

    EXAMPLES

    mate-desktop-item-edit --create-new ~/Desktop/Icon.desktop

    Open a mate-desktop-item-edit dialog to create a launcher icon on the desktop named "Icon".
