    NAME

    mGridExec - Start the execution of the mosaic on the grid

    SYNOPSIS

    mGridExec [-d] zipfile

    DESCRIPTION

    Given the name of the zip file, it starts the execution of the mosaic on the grid.

    BUGS

    This module have been used extensively but have not been subject to the same rigorous testing to which the core modules have been subjected. We advise users to contact Montage (montage@ipac.caltech.edu) before employing them to verify limitations in their use and changes in the interfaces.
