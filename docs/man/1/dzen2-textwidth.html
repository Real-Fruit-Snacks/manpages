    NAME

    dzen2-textwidth - calculate the width in pixels of text for a given font.

    SYNOPSIS

    dzen2-textwidth fontname string

    DESCRIPTION

    textwidth is a simple utility to calculate the width in pixels of text with a given font.

    This tool is provided as part of the dzen2 (1) package.

    OPTIONS

    A summary of options is included below.

    fontname
    Set the font.
    string
    Set the string.

    AUTHOR
