    NAME

    guess-helper - Tries to guess the configuration by comparing four dumps of the NVRAM.

    SYNOPSIS

    guess-helper

    DESCRIPTION

    guess-helper is a helper program from the package nvram-wakeup. It will guide you through the procedure of creating the files needed for the program guess and eventually will run guess at the end. The output will be a configuration file which might even work ;-) Don't change any other BIOS settings you are not told to change until the procedure is finished.

    See README.mb for more details.

    SEE ALSO

    nvram-wakeup(8), guess(8), README.mb
