    soundinfo(1) Linux System Manual soundinfo(1)

    NAME

    soundinfo - Describes parts of the Linux Sound system support

    SYNOPSIS

    soundinfo [options]

    DESCRIPTION

    soundinfo , describes parts of the Linux Sound system support in the current kernel. Output is to stdout and you may wish to pipe the output thru' your favorite pager.

    Currently only the capabilities and features of /dev/mixer and /dev/dsp are displayed. If your devices have different names (e.g. you have more than one sound card - or you are just plain perverse :-) then edit the source file and recompile.

    OPTIONS

    -h
    display usage and help info
