    NAME

    cpg_iteration_finalize - Finalize iterator for members of CPG

    SYNOPSIS

    #include <corosync/cpg.h>

    cs_error_t cpg_iteration_finalize (cpg_iteration_handle_t handle);

    DESCRIPTION

    The cpg_iteration_finalize function is used to free up memory associated with iteration obtained by calling of cpg_iteration_initialize(3) function. The handle argument is iterator handle obtained by cpg_iteration_initialize(3) function.

    RETURN VALUE

    This call returns the CS_OK value if successful. CS_ERR_BAD_HANDLE error is returned when handle is invalid.
