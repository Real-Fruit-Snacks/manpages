    NAME

    Net::LDAP::Intermediate::SyncInfo - LDAPv3 Sync Info Message object

    SYNOPSIS

    See Net::LDAP::Control::SyncRequest

    DESCRIPTION

    "Net::LDAP::Intermediate::SyncInfo" provides an interface for the creation and manipulation of objects that represent the "Sync Info Message" as described by RFC 4533.
