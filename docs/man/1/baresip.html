    BARESIP(1) User Commands BARESIP(1)

    NAME

    baresip - portable and modular command-line SIP user-agent

    SYNOPSIS

    baresip [options]

    DESCRIPTION

    baresip v1.0.0 Copyright (C) 2010 - 2020 Alfred E. Heggestad et al.

    options:

    -4
    Force IPv4 only
    -6
    Force IPv6 only
    -d
    Daemon
    -e <commands>
    Execute commands (repeat)
    -f <path>
    Config path
    -m <module>
    Pre-load modules (repeat)
    -p <path>
    Audio files
    -h -?
    Help
    -s
    Enable SIP trace
    -t <sec>
    Quit after <sec> seconds
    -n <net_if>
    Specify network interface
    -u <parameters>
    Extra UA parameters
    -v
    Verbose debug
