    NAME

    etapro - Driver for ETA UPS equipment

    NOTE

    This man page only documents the hardware-specific features of the etapro driver. For information about the core driver, see nutupsdrv(8).

    SUPPORTED HARDWARE

    This driver supports ETA UPS equipment with the "PRO" option for smart mode.

    EXTRA ARGUMENTS

    This driver does not support any extra settings in the ups.conf(5).
