    FFI(3) Library Functions Manual FFI(3)

    NAME

    FFI — Foreign Function Interface

    LIBRARY

    libffi, -lffi

    SYNOPSIS

    #include <ffi.h>

    ffi_status
    ffi_prep_cif(ffi_cif *cif, ffi_abi abi, unsigned int nargs, ffi_type *rtype, ffi_type **atypes);

    void
    ffi_prep_cif_var(ffi_cif *cif, ffi_abi abi, unsigned int nfixedargs, unsigned int ntotalargs, ffi_type *rtype, ffi_type **atypes);

    void
    ffi_call(ffi_cif *cif, void (*fn)(void), void *rvalue, void **avalue);

    DESCRIPTION

    The foreign function interface provides a mechanism by which a function can generate a call to another function at runtime without requiring knowledge of the called function's interface at compile time.
