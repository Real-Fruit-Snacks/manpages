    NAME

    libpipewire-module-fallback-sink - Fallback Sink

    DESCRIPTION

    Fallback sink, which appear dynamically when no other sinks are present.

    This is only useful for Pulseaudio compatibility.

    MODULE NAME

    libpipewire-module-fallback-sink

    MODULE OPTIONS

    • sink.name: sink name
    • sink.description: sink description
