    NAME

    apertium-compile-caps — capitalization restoration rule compiler for Apertium

    SYNOPSIS

    apertium-postchunk rule_file output_file

    DESCRIPTION

    This is the compiler for capitalization restoration (.crx) files.

    OPTIONS

    FILES

    These are the kinds of files that can be used with this command:

    rule_file
    A rules file with extension .crx.
    output_file
    A file with extension .crx.bin that will hold the result of compiling the rule_file file.
