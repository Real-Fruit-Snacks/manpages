    NAME

    naco_img_zpoint - Zero point computation recipe

    SYNOPSIS

    esorex naco_img_zpoint [OPTIONS] FILE.sof

    DESCRIPTION

    naco_img_zpoint -- Zero point recipe The files listed in the Set Of Frames (sof-file) must be tagged: NACO-raw-file.fits CAL_STD_CHOP or NACO-raw-file.fits CAL_STD_JITTER.

    NACO-Imaging-Standard-Star-Catalog.fits IMG_STD_CATALOG Optionally, a flat field frame may be inluded: NACO-flat-file.fits MASTER_IMG_FLAT

    OPTIONS

    --star_r <float>
    The star radius [arcsecond] (float; default: 2.0). The full name of this option for the EsoRex configuration file is naco.naco_img_zpoint.star_r [default = 2.0].
    --bg_r1 <float>
    The internal radius of the background [arcsecond] (float; default: 2.0). The full name of this option for the EsoRex configuration file is naco.naco_img_zpoint.bg_r1 [default = 2.0].
    --bg_r2 <float>
    The external radius of the background [arcsecond] (float; default: 3.0). The full name of this option for the EsoRex configuration file is naco.naco_img_zpoint.bg_r2 [default = 3.0].
    --ra <float>
    Right Ascension [Degrees] (float; default: 999.0). The full name of this option for the EsoRex configuration file is naco.naco_img_zpoint.ra [default = 999.0].
    --dec <float>
    DEClination [Degrees] (float; default: 999.0). The full name of this option for the EsoRex configuration file is naco.naco_img_zpoint.dec [default = 999.0].
    --pscale <float>
    Pixel scale (float; default: -1.0). The full name of this option for the EsoRex configuration file is naco.naco_img_zpoint.pscale [default = -1.0].
    --mag <float>
    Magnitude (float; default: 99.0). The full name of this option for the EsoRex configuration file is naco.naco_img_zpoint.mag [default = 99.0].
    --sx <int>
    Size of the search window in X-direction [pixel] (int; default: 10). The full name of this option for the EsoRex configuration file is naco.naco_img_zpoint.sx [default = 10].
    --sy <int>
    Size of the search window in Y-direction [pixel] (int; default: 10). The full name of this option for the EsoRex configuration file is naco.naco_img_zpoint.sy [default = 10].
    --check_im <bool>
    Create the check image (bool; default: False). The full name of this option for the EsoRex configuration file is naco.naco_img_zpoint.check_im [default = False].

    Note that it is possible to create a configuration file containing these options, along with suitable default values. Please refer to the details provided by the 'esorex --help' command.
