    NESTOPIA(6) Games Manual NESTOPIA(6)

    NAME

    nestopia - Nintento Entertainment System / Famicom emulator

    SYNOPSIS

    nestopia game

    DESCRIPTION

    This manual page documents briefly the nestopia command.

    nestopia is an emulator for the Nintento Entertainment System. It can execute programs intended for the NES console.

    OPTIONS

    nestopia has no command-line options; use the provided GUI to configure it.
