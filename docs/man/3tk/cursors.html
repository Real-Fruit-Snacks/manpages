    NAME
    

    cursors - mouse cursors available in Tk
    

    

    

    

    DESCRIPTION
    

    The -cursor widget option allows a Tk programmer to change
    the mouse cursor for a particular widget. The cursor names recognized by Tk
    on all platforms are:
