    NAME

    lebiniou - A tool to create images from sound

    SYNOPSIS

    lebiniou [OPTIONS]

    DESCRIPTION

    Le Biniou is a program which creates images from sound, in an intelligent way.

    OPTIONS

    -a | --audio file: Set audio input file (for the sndfile input plugin)

    -b | --basedir path: Set base directory to look for plugins (default: /usr/lib/lebiniou/plugins)

    -c | --config file: Select the configuration file to use (default: ~/.lebiniou/lebiniou.json)

    -d | --datadir path: Set data directory (default: /usr/share/lebiniou)

    -f | --fullscreen: Set fullscreen

    -h | --help: Display this help and exit

    -i | --input plugin: Set input plugin (inputs: sndio,alsa,jackaudio,oscaudio,pulseaudio,sndfile,twip,random,NULL)

    -m | --max-fps fps: Set maximum framerate

    -n | --no-borders: Disable window borders

    -o | --output plugin: Set output plugin. (outputs: SDL2,RTMP,mp4,v4l2loopback,NULL)

    -q | --quiet: Only display important messages to console

    -r | --random mode: Set auto-random mode

    -s | --soft-timers: Use soft timers instead of the system clock

    -t | --themes themes: Comma-separated list of themes to use

    -v | --version: Display the version and exit

    -w | --http-port port: Set HTTP API port

    -x | --width width: Set width

    -y | --height height: Set height

    -z | --schemes file: Set schemes file (default: /usr/share/lebiniou/etc/schemes.json)

    -B | --no-browser: Don't open the web interface in a browser

    -C | --cam-flip h|v: Flip webcam image horizontally/vertically

    -D | --device device: Webcam base (default: /dev/video)

    -E | --start-encoding: Start encoding directly

    -F | --fade delay: Set fading delay (s)

    -I | --instance name: Set the instance name

    -N | --no-statistics: Do not send usage statistics

    -P | --playlist file: Set playlist file

    -S | --stats: Display statistics and exit

    -W | --webcams webcams: Number of webcams (default: 1)

    -X | --xpos x position: Set left position of the window

    -Y | --ypos y position: Set top position of the window
