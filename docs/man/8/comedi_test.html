    NAME

    comedi_test - Program to test comedilib

    SYNOPSIS

    comedi_test [options]

    DESCRIPTION

    This manual page documents briefly the comedi_test command. This manual page was written for the Debian distribution because the original program does not have a manual page.

    OPTIONS

    These programs follow the usual GNU command line syntax, with long options starting with two dashes (`-'). A summary of options is included below.

    -f <device_file> or --device <device_file>
    Use device <device_file> (/dev/comedi0).
    -S <arg> or --save-file <arg>
    Save calibration file
    --help
    Show help and exit.
