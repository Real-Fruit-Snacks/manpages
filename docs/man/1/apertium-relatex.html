    APERTIUM-RELATEX(1) General Commands Manual APERTIUM-RELATEX(1)

    NAME

    apertium-relatex — LaTeX format processor for Apertium

    SYNOPSIS

    apertium-relatex [input_file [output_file]]

    DESCRIPTION

    apertium-relatex This filter preprocess apertium generator output and removes superblanks marks to do subsequent processing in apertium pipeline.

    OPTIONS

    -h, --help
    Display this help.

    SEE ALSO

    apertium(1), apertium-deslatex(1), apertium-destxt(1), apertium-postlatex(1), apertium-postlatex-raw(1), apertium-prelatex(1), lt-proc(1)
