    USB-CREATOR-KDE(8) Ubuntu Startup Disk Creator For KDE USB-CREATOR-KDE(8)

    NAME

    usb-creator-kde ‐ Ubuntu startup disk creation tool for KDE

    SYNOPSIS

    usb-creator-kde [-i|--iso <img>]
    [-h|--help] [-v|--version]

    DESCRIPTION

    usb-creator-kde takes an Ubuntu CD image file as input and writes it out in a usable form to a removable disk.

    OPTIONS

    -i, --iso <img>
    Provide a source image to pre‐populate the UI.
    -h,--help
    Print a summary of command line options and exit.
    -v,--version
    Print the current version and exit.

    AUTHOR

    REPORTING BUGS

    Report bugs via Ubuntu Launchpad <https://launchpad.net/ubuntu/+source/usb-creator/+filebug>

    When reporting bugs, please be sure to include a copy of the usb-creator log file to assist in debugging. This log file can be found as ~/.cache/usb-creator.log.
