    NAME

    asn1_find_structure_from_oid - API function

    SYNOPSIS

    #include <libtasn1.h>

    const char * asn1_find_structure_from_oid(asn1_node_const definitions, const char * oidValue);

    ARGUMENTS

    asn1_node_const definitions
    ASN1 definitions
    const char * oidValue
    value of the OID to search (e.g. "1.2.3.4").

    DESCRIPTION

    Search the structure that is defined just after an OID definition.

    RETURNS

    NULL when oidValue not found, otherwise the pointer to a constant string that contains the element name defined just after the OID.
