    NAME

    pdp-config — script to get information about the installed version of PDP

    SYNOPSIS

    pdp-config [--version] [--cflags] [--libdir]

    DESCRIPTION

    pdp-config is a tool that is used to configure to determine the compiler and linker flags that should be used to compile and links modules that use PDP.

    OPTIONS

    pdp-config accepts the following options:

    --version
    Print the currently installed version of PDP on the standard output.
    --cflags
    Print the compiler flags that are necessary to compile a PDP module.
    --libdir
    Print the linker flags that are necessary to compile a PDP module.
