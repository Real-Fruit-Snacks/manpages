    NAME

    debget - Fetch one or more .deb files for a package in APT's database

    SYNOPSIS

    debget package[=version] [package>[=version] ...]

    DESCRIPTION

    debget fetches .deb files for one or more packages from an Debian mirror. It uses apt-get(8)'s download subcommand to download the package so it will download the same version from the same server as apt-get(8)'s install subcommand would do. The .deb file will be saved into your current working directory.

    If a "=version" with a package version number is appended to a package name, that package version (if available) will be downloaded.
