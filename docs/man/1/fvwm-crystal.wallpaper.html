    NAME

    fvwm-crystal.wallpaper - random wallpaper selector

    SYNOPSIS

    fvwm-crystal.wallpaper [directory] [directory] [...]

    DESCRIPTION

    fvwm-crystal.wallpaper is a program for choosing random wallpapers. It is shipped with the fvwm-crystal desktop environment. From a list of directories, it randomly chooses one wallpaper and returns its filename on stdout.

    This manual page was written for the Debian distribution because the original program does not have a manual page.
