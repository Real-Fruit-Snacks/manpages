    QMC:(1) User Commands QMC:(1)

    NAME

    QMC: - simplification tool (by using the Quine - McClusky process)

    SYNOPSIS

    qmc [OPTION]

    DESCRIPTION

    Simplification tool by using the Quine - McClusky process.

    OPTIONS

    -a, --avoidast
    avoid astericks at the output

    -d, --describesyntax descibe the syntax for the EQU

    -g, --gettable
    get boolean table
    -G --gethtmltable
    get boolean table as HTML output
    -h, --help
    display this help and exit
    -p, --prompt
    prompt for equation
    -s, --source EQU
    set one side of the equation
    -t, --table
    set boolean table
    -T, --time
    display used time
    -v, --verbose
    print progress
    --version
    output version information and exit

    A few samples:

    qmc -s "/[a+b]*[/b*c]" qmc -g -s "x + //y * [ z + /z * y ] * 1" qmc -T -s "a*/b+c+d+e*/c+f*e" qmc -p
    (symbols are recognized as single char's)
