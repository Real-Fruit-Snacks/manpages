    NAME

    reduce-font - reduce a font to a set of glyphs

    SYNOPSIS

    reduce-font font.bdf < text

    DESCRIPTION

    reduce-font reduces the content of a font, provided the set of text that will have to be displayed with the reduced font. There is no option.
