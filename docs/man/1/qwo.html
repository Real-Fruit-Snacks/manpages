    NAME

    qwo - An input method for touchscreen

    SYNOPSIS

    qwo [options]

    OPTIONS

    -g, --geometry [size][{+-}<xoffset>[{+-}<yoffset>]]
    specify window size and/or position
    -c, --config <file>
    use configuration file <file> instead of ~/.qworc
    -f, --foreground <color>
    set the foreground color
    -b, --background <color>
    set the background color

    -d, --delimiter-color <color> set the region delimiter color

    Where color is one of 0xrrggbb or a named color as red...
    -h, --help
    Print this help
    -v, --version
    Print version information
