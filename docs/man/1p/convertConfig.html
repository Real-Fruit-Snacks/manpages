    CONVERTCONFIG(1p) User Contributed Perl Documentation CONVERTCONFIG(1p)

    NAME

    convertConfig - A tool to migrate Lemonldap::NG configuration database to a new backend.

    SYNOPSIS

    convertConfig [--help] [--current /old/lemonldap-ng.ini] [--new /new/lemonldap-ng.ini] [--latest] [--overwrite] [--force]
    Convert from default File backend to the currently configured backend
    convertConfig
    Convert only the latest config (configuration history is lost)
    convertConfig --latest
    Specify different INI config files containing old or new backends
    convertConfig --current config --new config
    Continue even if an error occurs
    convertConfig --force

    DESCRIPTION

    convertConfig is a command line tool to convert all configurations stored in database and initialize a new Lemonldap::NG configuration based on current configuration.
