    NAME

    ipdiscover - tool to allow OCS Inventory NG to discover network connected devices

    DESCRIPTION

    IP discovery feature allow OCS Inventory NG to discover all network connected devices on the network.

    For this, Communication server asks a number of most XactiveX computers running OCS Inventory NG agent to scan for MAC addresses in their sub network at each run. They will not scan the entire wide area network, but only their local network defined by the couple IP address/subnet mask.
