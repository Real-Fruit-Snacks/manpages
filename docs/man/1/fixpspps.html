    NAME

    fixpspps - filter to fix PSPrint PostScript so PSUtils work

    SYNOPSIS

    fixpspps < PSPrint.ps > Fixed.ps

    DESCRIPTION

    Fixpspps is a perl filter which "fixes" PostScript from PSPrint so that it works correctly with Angus Duggan's psutils package.
