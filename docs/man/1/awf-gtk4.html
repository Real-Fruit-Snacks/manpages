    NAME

    awf-gtk4 — Theme preview application for GTK 4.

    SYNOPSIS

    awf-gtk4 [-h] [-v] [-l] [-n] [-t theme] [-s filename] [--ltr] [--rtl]

    DESCRIPTION

    A widget factory is a theme preview application for GTK. It displays the various widget types provided by GTK in a single window allowing to see the visual effect of the applied theme.

    -h
    Show help.
    -v
    Show version number.
    -l
    List available themes.
    -n
    Don't start spinners.
    -t theme
    Run with the specified theme.
    -s filename
    Run and save a png screenshot on SIGHUP.
    --ltr
    Start with text from left to right (Left-To-Right).
    --rtl
    Start with text from right to left (Right-To-Left).
