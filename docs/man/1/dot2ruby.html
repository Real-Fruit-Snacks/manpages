    DOT2RUBY(1) DOT2RUBY(1)

    NAME

    dot2ruby - create a ruby script from a graphviz script

    SYNOPSIS

    dot2ruby [-ofile] [-Tformat] [-h] [-V] script

    DESCRIPTION

    dot2ruby is a tool that allows you to create a ruby script from a graphviz script.

    See </usr/share/doc/ruby-graphviz/> for more details.

    OPTIONS

    -o, --output-file [file]
    Path to output image file (default STDOUT)
    -T, --output-format [format]
    Output format (default: png)
    -p, --path
    Graphviz path
    -V, --version
    Show version
    -h, --help
    Show this usage message

    EXAMPLE

    $ cat hello.dot
digraph G {Hello->World;}
$ dot2ruby hello.dot
