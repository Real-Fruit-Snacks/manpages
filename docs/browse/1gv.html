    section 1gv — User commands

    • animate(1gv) animate a sequence of geomview files
    • anytooff(1gv) filter to convert a (set of) geomview data file(s) containing any geomview primitives into a single polylist.
    • anytoucd(1gv) filter to convert a geomview data file into a data file in the so-called 'ucd' (Unstructured Cell Data) format.
    • bdy(1gv) boundary curve finder
    • bez2mesh(1gv) dices a BEZ file to a list of MESHes
    • clip(1gv) Clip an OOGL object against planes or other surfaces
    • geomview(1gv) interactive geometry viewer
    • hvectext(1gv) construct Geomview VECT text object from Hershey fonts
    • math2oogl(1gv) convert Mathematica grphics object to OOGL format
    • nose(1gv) test geomview's picking facility
    • offconsol(1gv) polylist consolidator.
    • oogl2rib(1gv) convert OOGL file to RenderMan rib format
    • oogl2vrml(1gv) convert OOGL to VRML
    • polymerge(1gv) merge coincident vertices, collinear edges, coplanar faces in an OOGL OFF object
    • togeomview(1gv) send commands or OOGL objects to geomview
    • ucdtooff(1gv) filter to convert an AVS 'ucd' (Unstructured Cell Data) format file into an OFF file.
    • vrml2oogl(1gv) convert VRML to OOGL
