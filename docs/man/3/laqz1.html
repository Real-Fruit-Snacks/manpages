    CLAQZ1

    Purpose:


    
      CLAQZ1 chases a 1x1 shift bulge in a matrix pencil down a single position

    Parameters

    ILQ 


    
          ILQ is LOGICAL

    
              Determines whether or not to update the matrix Q

    ILZ


    
          ILZ is LOGICAL

    
              Determines whether or not to update the matrix Z

    K


    
          K is INTEGER

    
              Index indicating the position of the bulge.

    
              On entry, the bulge is located in

    
              (A(k+1,k),B(k+1,k)).

    
              On exit, the bulge is located in

    
              (A(k+2,k+1),B(k+2,k+1)).

    ISTARTM


    
          ISTARTM is INTEGER

    ISTOPM


    
          ISTOPM is INTEGER

    
              Updates to (A,B) are restricted to

    
              (istartm:k+2,k:istopm). It is assumed

    
              without checking that istartm <= k+1 and

    
              k+2 <= istopm

    IHI


    
          IHI is INTEGER

    LDA


    
          LDA is INTEGER

    
              The leading dimension of A as declared in

    LDB


    
          LDB is INTEGER

    
              The leading dimension of B as declared in

    
              the calling procedure.

    NQ


    
          NQ is INTEGER

    
              The order of the matrix Q

    QSTART


    
          QSTART is INTEGER

    
              Start index of the matrix Q. Rotations are applied

    Q

    LDQ

    DLAQZ1

    Purpose:


    
      Given a 3-by-3 matrix pencil (A,B), DLAQZ1 sets v to a

    
      scalar multiple of the first column of the product

    
      (*)  K = (A - (beta2*sr2 - i*si)*B)*B^(-1)*(beta1*A - (sr2 + i*si2)*B)*B^(-1).

    
      It is assumed that either

    
              1) sr1 = sr2

    
          or

    
              2) si = 0.

    
      This is useful for starting double implicit shift bulges

    
      in the QZ algorithm.

    Parameters

    A 


    
          A is DOUBLE PRECISION array, dimension (LDA,N)

    
              The 3-by-3 matrix A in (*).

    LDA


    
          LDA is INTEGER

    
              The leading dimension of A as declared in

    
              the calling procedure.

    B


    
          B is DOUBLE PRECISION array, dimension (LDB,N)

    
              The 3-by-3 matrix B in (*).

    LDB


    
          LDB is INTEGER

    
              The leading dimension of B as declared in

    
              the calling procedure.

    SR1


    
          SR1 is DOUBLE PRECISION

    SR2


    
          SR2 is DOUBLE PRECISION

    SI


    
          SI is DOUBLE PRECISION

    BETA1


    
          BETA1 is DOUBLE PRECISION

    BETA2


    
          BETA2 is DOUBLE PRECISION

    V


    
          V is DOUBLE PRECISION array, dimension (N)

    
              A scalar multiple of the first column of the

    
              matrix K in (*).
