    NAME

    gnome-mastermind - A Mastermind clone for GNOME

    SYNOPSIS

    gnome-mastermind [OPTIONS]...

    DESCRIPTION

    This manual page documents briefly the GNOME Mastermind game, a Mastermind clone for GNOME.

    OPTIONS

    gnome-mastermind accepts only the standard GApplication/GTK+ options. Try “gnome-mastermind --help-all” to obtain more information about these options.

    ENVIRONMENT

    GM_DEBUG
    If set to “yes”, the program will print extra messages on the console, occasionally useful for debugging purposes. Normally you wouldn't need this unless you are trying to fix a bug.
