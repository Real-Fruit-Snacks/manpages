    NAME

    bestfcom - Driver for Best Power Fortress/Ferrups

    NOTE

    This man page only documents the hardware-specific features of the bestfcom driver. For information about the core driver, see nutupsdrv(8).

    SUPPORTED HARDWARE

    Best Power Fortress/Ferrups implementing the Fortress UPS Protocol (f-command set). (For older Fortress units, see bestfortress(8).)

    EXTRA ARGUMENTS

    This driver does not support any extra settings in the ups.conf(5).
