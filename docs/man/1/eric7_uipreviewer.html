    NAME

    eric7_uipreviewer - Previewer for Qt Designer *.ui files

    SYNOPSIS

    eric7_uipreviewer [options] files...

    DESCRIPTION

    This manual page documents briefly the eric7_uipreviewer command. This manual page was written for the Debian distribution because the original program does not have a manual page.

    eric7_uipreviewer is a small preview program for user interface files created with Qt Designer.

    OPTIONS

    These programs follow the usual GNU command line syntax, with long options starting with two dashes (`-'). A summary of options is included below. For a complete description, see the Info files.

    -h, --help
    Show summary of options.
    -v, --version
    Show version of program.
