    AM7XXX-MODESWITCH(1)   AM7XXX-MODESWITCH(1)

    NAME

    am7xxx-modeswitch - change the operational mode of am7xxx based devices

    SYNOPSIS

    am7xxx-modeswitch

    DESCRIPTION

    am7xxx-modeswitch(1) is a minimal replacement of usb-modeswitch to use with am7xxx devices (e.g. Acer C110 or Philips PPX projectors) to switch from the mass storage device mode to the generic display mode.

    It is handy on systems where usb-modeswitch is not available, like Windows.

    EXAMPLE OF USE

    am7xxx-modeswitch

    EXIT STATUS

    0

    Success

    !0

    Failure (libusb error)
