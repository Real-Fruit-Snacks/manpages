    NAME

    XtName - obtain widget's name

    SYNTAX

    #include <X11/Intrinsic.h>

    String XtName(Widget w);

    ARGUMENTS

    w
    Specifies the widget.

    DESCRIPTION

    XtName returns the widget's name.
