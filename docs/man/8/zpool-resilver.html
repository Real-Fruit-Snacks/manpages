    NAME

    zpool-resilver — resilver devices in ZFS storage pools

    SYNOPSIS

    zpool resilver pool…

    DESCRIPTION

    Starts a resilver of the specified pools. If an existing resilver is already running it will be restarted from the beginning. Any drives that were scheduled for a deferred resilver will be added to the new one. This requires the resilver_defer pool feature.
