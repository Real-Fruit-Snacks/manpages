    NAME

    guix - manual page for guix publish (GNU Guix) 1.3.0.27848-a3264

    SYNOPSIS

    guix publish [OPTION]...

    DESCRIPTION

    Publish /gnu/store over HTTP.

    -p, --port=PORT
    listen on PORT
    --listen=HOST
    listen on the network interface for HOST
    -u, --user=USER
    change privileges to USER as soon as possible
    -a, --advertise
    advertise on the local network
    -C, --compression[=METHOD:LEVEL]
    compress archives with METHOD at LEVEL
    -c, --cache=DIRECTORY
    cache published items to DIRECTORY
    --cache-bypass-threshold=SIZE
    serve store items below SIZE even when not cached
    --workers=N
    use N workers to bake items
    --ttl=TTL
    announce narinfos can be cached for TTL seconds

    --negative-ttl=TTL announce missing narinfos can be cached for TTL seconds

    --nar-path=PATH
    use PATH as the prefix for nar URLs
    --public-key=FILE
    use FILE as the public key for signatures

    --private-key=FILE use FILE as the private key for signatures

    -r, --repl[=PORT]
    spawn REPL server on PORT
    -h, --help
    display this help and exit
    -V, --version
    display version information and exit
