    NAME

    rst-buildhtml - convert many reST documents to HTML View document source. Generated on: 2023-11-30 11:16 UTC. Generated by Docutils from reStructuredText source.

    SYNOPSIS

    rst-buildhtml [options] [<directory> ...]

    DESCRIPTION

    Generates .html from all the reStructuredText .txt files (including PEPs) in each <directory> (default is the current directory).
