    NAME

    byobu-janitor - script for cleaning and upgrading environment after upgrades

    SYNOPSIS

    byobu-janitor

    DESCRIPTION

    byobu-janitor is script for cleaning environment after upgrades, it consists from several tasks where aim is to ensure that environment is ready for new version of byobu.
