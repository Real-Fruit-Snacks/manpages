    NAME

    dpt-missing-pristine-tar - fix missing pristine-tar information

    SYNOPSIS

    dpt missing-pristine-tar

    DESCRIPTION

    dpt missing-pristine-tar fixes missing pristine-tar(1) information by downloading and importing upstream sources.

    The download first tries apt-get source and if that fails, falls back to uscan(1).
