    NAME

    pkegcs - Utility for raster files in European Grid Coordinate System

    SYNOPSIS


    pkegcs
    [options]

    DESCRIPTION

    pkegcs is a utility for raster files in European Grid Coordinate System

    OPTIONS

    -i filename, --image filename
    input image to analyse
    -b band, --band band
    Band specific information
    -c2b value, --cell2bb value
    convert cell code to geo coordinates of boundingbox (e.g. 32-AB)
    -c2m value, --cell2mid value
    convert cell code to centre in geo coordinates (e.g. 32-AB)
    -g2c, --geo2cell
    get cell code for coordinates in -x and -y given the resolution in -dx
    -ref, --ref
    get reference pixel (lower left corner of centre of gravity pixel)
    -m mask, --mask mask
    mask value(s) for no data to calculate reference pixel in image
    -dx number, --dx number
    resolution (default: 250)
    -x coord, --x coord
    x coordinate in epsg:3035
    -y coord, --y coord
    y coordinate in epsg:3035
