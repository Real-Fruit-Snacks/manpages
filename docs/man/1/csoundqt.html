    CSOUNDQT(1) General Commands Manual CSOUNDQT(1)

    NAME

    csoundqt - GUI interface and IDE for the csound synthesis system

    SYNOPSIS

    csoundqt [files]

    DESCRIPTION

    This manual page documents briefly the csoundqt command.

    CsoundQT is a simple cross platform editor and front-end for Csound with syntax highlighting, interactive help and automatic launching of Csound
