    NAME

    condor_submit_makeflow - submit workflow to HTCondor batch system

    SYNOPSIS

    condor_submit_makeflow [options] <workflow>

    DESCRIPTION

    condor_submit_makeflow submits Makeflow itself as a batch job, allowing HTCondor to monitor and control the job, so the user does not need to remain logged in while the workflow runs. All options given to condor_submit_makeflow are passed along to the underlying Makeflow.

    EXIT STATUS

    On success, returns zero. On failure, returns non-zero.

    EXAMPLE

    condor_submit_makeflow example.mf
Submitting example.mf as a background job to HTCondor...
Submitting job(s).
1 job(s) submitted to cluster 364.
Makeflow Output : makeflow.364.output
Makeflow Error  : makeflow.364.error
Condor Log      : makeflow.364.condorlog
