    NAME

    polygen-data - assorted polygen grammars

    SYNOPSIS

    polygen grammar

    DESCRIPTION

    All of these grammars output all sort of strange and out-of-control things. No personal, racial, societal slurs are intended. For amusement only.

    You can find out more information about a grammar with the command polygen -info grammar
