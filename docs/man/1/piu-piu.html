    NAME

    piu-piu-1.0 - BASH game.

    DESCRIPTION

    This is an Old School horisontal scroller 'Shoot Them All' game in bash. With co-op mode. You have to defeat 100 aliens to fight with Boss. To play in co-op mode first, start the server, then start the client. Terminals on both hosts have to be with equal dimensions.

    CONTROLS

    w, W
    Fly up.
    a, A
    Fly left.
    s, S
    Fly down.
    d, D
    Fly right.
    p, P
    Shoot or select.

    Ivan Marov <ie.marov@gmail.com>
