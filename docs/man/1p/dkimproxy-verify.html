    NAME

    dkimverify.pl - verifies DKIM signatures on an email message

    SYNOPSIS

      dkimverify.pl [options] < signed_email.txt
    options:
      --debug-canonicalization=FILE
  dkimverify.pl --help
    to see a full description of the various options
