    template<typename _IntType = int>

    class std::negative_binomial_distribution< _IntType >"A negative_binomial_distribution random number distribution.

    The formula for the negative binomial probability mass function is $p(i) = } p^i (1 - p)^{t - i}$ where $t$ and $p$ are the parameters of the distribution.
