    NAME

    pyscss - Converts Scss files to CSS.

    SYNOPSIS

    pyscss [-h] [--help]

    DESCRIPTION

    Run pyscss --help for more options.

    EXAMPLE

    	pyscss --no-compress input.scss > output.css
