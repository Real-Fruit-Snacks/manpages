    NAME

    pngtobogl - Convert PNG image to bogl C file

    SYNOPSIS

    pngtobogl graphic.png > graphic.c

    DESCRIPTION

    pngtobogl converts a PNG image into a C file that can be used with the bogl library. There is no option.
