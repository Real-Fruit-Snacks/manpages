    nbdkit-ones-plugin(1) NBDKIT nbdkit-ones-plugin(1)

    NAME

    nbdkit-ones-plugin - nbdkit plugin filled with repeated 0xff or other bytes

    SYNOPSIS

     nbdkit ones [[size=]SIZE] [byte=N]

    DESCRIPTION

    "nbdkit-ones-plugin" is a plugin for nbdkit(1). Any read returns a repeating pattern of 0xff (all ones) bytes, or another repeating byte if you use the "byte" parameter. Everything written to the virtual device is discarded.

    Most NBD consumers will be very confused by this device. It is mainly useful for testing. nbdkit-data-plugin(1) can be used to create a modifiable RAM disk initialized with fixed data.

    PARAMETERS

    byte=N
    Set the repeating byte to "N". The default is 0xff (all ones).

    Instead of "byte=0" it is more efficient to use nbdkit-null-plugin(1).

    [size=]SIZE
    Specify the virtual size of the disk image.

    This parameter is optional. If omitted then a zero-sized device is created.

    "size=" is a magic config key and may be omitted in most cases. See "Magic parameters" in nbdkit(1).

    FILES

    $plugindir/nbdkit-ones-plugin.so
    The plugin.

    Use "nbdkit --dump-config" to find the location of $plugindir.

    VERSION

    "nbdkit-ones-plugin" first appeared in nbdkit 1.36.
