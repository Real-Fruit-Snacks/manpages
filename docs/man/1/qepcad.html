    NAME

    QEPCAD - Quantifier Elimination by Partial Cylindrical Algebraic Decomposition

    SYNOPSIS

    qepcad [-h|-v] [-noecho] [-t <num> ] [+N <numcells> ]

    DESCRIPTION

    QEPCAD B is a program for studying Cylindrical Algebraic Decomposition (CAD). It constructs CADs from an input formula and variable order, and provides many commands for getting info out of the CAD, including construction of simple equivalent Tarski formulas.

    OPTIONS

    -h
    Help information
    -v
    Print version string
    -noecho
    Turns off echoing of input
    -t <num>
    Sets a timeout of <num> seconds.
    +N<numcells>
    Sets garbage collected space to <numcells>. Default is 2000000. Maximum allowable is 1073741822 (Because "Word" is 32 bits).
    +h
    Prints Saclib help message.
