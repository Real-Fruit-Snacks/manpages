    NAME

    glewlwyd - Single-Sign-On (SSO) server with multiple factor authentication

    DESCRIPTION

    Glewlwyd Single-Sign-On (SSO) server with multiple factor authentication

    Command-line options:

    -c --config-file PATH

    Path to configuration file

    -e --env-variables

    Use environment variables to configure Glewlwyd

    -p --port PORT

    Port to listen to

    -u --url-prefix PREFIX

    API URL prefix

    -m --log-mode MODE

    Log modes available: console, syslog or file If you want multiple modes, separate them with a comma "," default: console

    -l --log-level LEVEL

    Log levels available: NONE, ERROR, WARNING, INFO, DEBUG default: INFO

    -f --log-file PATH

    Path for log file if log mode file is specified

    -v --version

    Print Glewlwyd's current version

    -h --help

    Print this message

    SEE ALSO

    The full documentation for glewlwyd is maintained as a Texinfo manual. If the info and glewlwyd programs are properly installed at your site, the command

    info glewlwyd

    should give you access to the complete manual.
