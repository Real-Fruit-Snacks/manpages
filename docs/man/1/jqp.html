    JQP(1) User Commands JQP(1)

    NAME

    jqp - a TUI to explore the jq command line utility

    DESCRIPTION

    jqp is a TUI to explore the jq command line utility

    Usage:

    jqp [flags]

    Flags:

    --config string
    path to config file (default is $HOME/.jqp.yaml)
    -f, --file string
    path to the input JSON file
    -h, --help
    help for jqp
    -t, --theme string
    jqp theme
    -v, --version
    version for jqp
