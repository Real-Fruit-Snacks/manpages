    NAME

    pipewire-filter-chain.conf - filter-chain.conf

    DESCRIPTION

    PipeWire example configuration for running audio filters.

    SYNOPSIS

    $XDG_CONFIG_HOME/pipewire/filter-chain.conf

    /etc/pipewire/filter-chain.conf

    /usr/share/pipewire/filter-chain.conf

    /usr/share/pipewire/filter-chain.conf.d/

    /etc/pipewire/filter-chain.conf.d/

    $XDG_CONFIG_HOME/pipewire/filter-chain.conf.d/

    DESCRIPTION

    When pipewire(1) is run using this configuration file, pipewire -c filter-chain.conf, it starts a PipeWire client application that publishes nodes that apply various audio filters to their input.

    It is a normal PipeWire client application in all respects.

    Drop-in configuration files filter-chain.conf.d/*.conf can be used to modify the filter configuration, see pipewire.conf(5). Some examples are in /usr/share/pipewire/filter-chain/
