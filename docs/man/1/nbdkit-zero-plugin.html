    NAME

    nbdkit-zero-plugin - nbdkit zero length plugin

    SYNOPSIS

     nbdkit zero

    DESCRIPTION

    "nbdkit-zero-plugin" is a zero length plugin for nbdkit(1). It presents a virtual device of length zero bytes.

    This plugin is mainly good for testing corner cases in nbdkit, the NBD protocol and NBD clients.

    FILES

    $plugindir/nbdkit-zero-plugin.so
    The plugin.

    Use "nbdkit --dump-config" to find the location of $plugindir.

    VERSION

    "nbdkit-zero-plugin" first appeared in nbdkit 1.4.
