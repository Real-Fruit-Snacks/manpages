    NAME

    lutefisk-doc - software for the de novo interpretation of peptide CID spectra (documentation)

    DESCRIPTION

    This manual page documents briefly the lutefisk-doc package. This package contains the user manual for the lutefisk software package. The documentation is provided in the form of a set of HTML files. The files are located in /usr/share/doc/lutefisk-doc. Note that the documentation is also available at http://www.hairyfatguy.com/lutefisk/docs/.
