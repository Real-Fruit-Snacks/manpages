    NAME

    fvwm-crystal.generate-menu - generate FVWM-Crystal menu

    SYNOPSIS

    fvwm-crystal.generate-menu [name]

    DESCRIPTION

    fvwm-crystal.generate-menu will generate FVWM-Crystal menu from desktop and icons files provided by the applications. If a name is provided as an argument only the menu entry for the provided application will be generated.

    This manual page was written for the Debian distribution because the original program does not have a manual page.
