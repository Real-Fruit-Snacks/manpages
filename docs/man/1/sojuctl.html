    NAME

    sojuctl - Manage a running instance of the soju IRC bouncer

    SYNOPSIS

    sojuctl [options...] <command...>

    DESCRIPTION

    sojuctl sends a BouncerServ command to a running soju instance. See the IRC SERVICE section in soju(1) for more information.

    sojuctl requires a listen unix+admin:// directive in the soju configuration file. sojuctl needs to be run with write permissions on the soju admin socket.

    OPTIONS

    -h, -help

    Show help message and quit.

    -config <path>

    Path to the config file. If unset, the default config file path is used, if any.

    AUTHORS
