    NAME

    backintime-askpass - a simple backup tool for Linux.

    This is the command line tool for piping passwords into ssh/sshfs and encfs.

    SYNOPSIS

    backintime-askpass

    DESCRIPTION

    Back In Time is a simple backup tool for Linux. This is a helper tool for piping passwords into ssh/sshfs and encfs. Options will will be read from environ variables. It doesn't provide any useful enduser service.

    ENVIRON

    ASKPASS_PROFILE_ID
    Back In Time Profile-ID.
    ASKPASS_MODE
    Backup mode (or backend). Take a look at 'man backintime-config' section profile<N>.snapshots.mode
    ASKPASS_TEMP
    Temp FIFO socket used to pipe the password

    SEE ALSO

    backintime, backintime-config.

    Back In Time also has a website: https://github.com/bit-team/backintime
