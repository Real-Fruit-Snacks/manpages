    NAME

    ocf-tester - Part of the Linux-HA project

    SYNOPSIS

    ocf-tester [-hvqdX] -n resource_name [-o name=value]* /full/path/to/resource/agent

    DESCRIPTION

    Tool for testing if a cluster resource is OCF compliant

    OPTIONS

    -v
