    NAME

    dothost - DNS lookup with Graphviz output

    SYNOPSIS

    dothost ADDRESS [ADDRESS...]

    DESCRIPTION

    dothost is a DNS lookup utility, which produces output in Graphviz format.

    OPTIONS

    -h, --help
    Show help message and exit.
    --version
    Show version information and exit.

    EXAMPLES

    $ dothost www.iana.org | graph-easy --as boxart > file.boxart
$ dothost www.iana.org | graph-easy --as html > file.html
$ dothost www.iana.org | graph-easy --as svg > file.svg
$ dothost www.iana.org | graph-easy --as dot > file.dot
$ dothost www.iana.org | graph-easy --as txt > file.txt
$ dothost www.iana.org | graph-easy --as vcg > file.vcg
$ dothost www.iana.org | graph-easy --as gdl > file.gdl
