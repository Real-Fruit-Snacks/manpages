    NAME

    wmslite - light-weight WMS server supporting RasterLite2 DataSources

    SYNOPSIS


    wmslite
    ARGLIST

    DESCRIPTION

    wmslite is a light-weight WMS server supporting RasterLite2 DataSources.

    OPTIONS

    -db pathname, --db-path pathname
    RasterLite2 DB path
    -p number, --ip-port number
    IP port number [default: 8080]
    -cs num, --cache-size num
    DB cache size (how many pages)
    -dbg, --debug
    verbose debugging mode
