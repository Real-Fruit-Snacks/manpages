    Detailed Description

    template<typename _Tp>

    class __gnu_cxx::__detail::__mini_vector< _Tp >"__mini_vector<> is a stripped down version of the full-fledged std::vector<>.

    It is to be used only for built-in types or PODs. Notable differences are:

    1.
    Not all accessor functions are present.
    2.
    Used ONLY for PODs.
    3.
    No Allocator template argument. Uses operator new() to get memory, and operator delete() to free it. Caveat: The dtor does NOT free the memory allocated, so this a memory-leaking vector!
