    ORTHANC(8) System Administration tools and Deamons ORTHANC(8)

    NAME

    Orthanc - Lightweight, RESTful DICOM server for healthcare and medical research

    SYNOPSIS

    OrthancRecoverCompressedFile [INPUT] [OUTPUT]

    DESCRIPTION

    Orthanc, lightweight, RESTful DICOM server for healthcare and medical research.

    This is a maintenance tool to recover a DICOM file that was compressed by Orthanc. If no output is given, the data will be output to stdout. Please refer to the Orthanc homepage for the full instructions about how to use Orthanc <http://www.orthanc-server.com/>.

    Exit status:

    0 if success,

    -1 if error.
