    NAME

    rst2latex - convert reST documents to LaTeX View document source. Generated on: 2023-11-30 11:16 UTC. Generated by Docutils from reStructuredText source.

    SYNOPSIS

    rst2latex [options] [<source> [<destination>]]

    DESCRIPTION

    Generates LaTeX documents from standalone reStructuredText sources. Reads from <source> (default is stdin) and writes to <destination> (default is stdout). See <https://docutils.sourceforge.io/docs/user/latex.html> for the full reference.
