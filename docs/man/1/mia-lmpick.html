    NAME

    mia-lmpick - 3D volume visualization and landmark picking

    SYNOPSIS

    mia-lmpick [OPTION...]

    DESCRIPTION

    mia-lmpick is a tool to visualize iso-surfaces in volume data and to pick landmarks in these data sets. For a tutorial see https://sourceforge.net/p/mia/wiki/lmpick/

    OPTIONS

    Application options

    -i, --landmarkset=landmarkset
    load an initial landmark set
    -o, --outputset=outputset
    set predefined output landmark set file
    -v, --volume=volume
    load volume image (in Vista format)

    Help options

    -?, --help
    Show this help message
    --usage
    Display brief usage message
