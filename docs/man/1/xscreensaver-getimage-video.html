    NAME

    xscreensaver-getimage-video - grab a video still frame for use by screen savers

    SYNOPSIS

    xscreensaver-getimage-video [--verbose]

    DESCRIPTION

    The xscreensaver-getimage-video program is a helper program for the xscreensaver hacks that manipulate images. Specifically, it is invoked by xscreensaver-getimage(6x) as needed.

    It grabs a single frame of video from the system's video input to a file, and prints the name of that file on stdout. It does this by figuring out which frame-grabbing programs are installed on the system, and invoking the first one it finds.

    OPTIONS

    xscreensaver-getimage-video accepts the following options:

    --verbose
    Print diagnostics.
