    NAME

    TAP::Parser::Result::YAML - YAML result token.

    VERSION

    Version 3.44

    DESCRIPTION

    This is a subclass of TAP::Parser::Result. A token of this class will be returned if a YAML block is encountered.

     1..1
 ok 1 - woo hooo!

    1..1 is the plan. Gotta have a plan.
