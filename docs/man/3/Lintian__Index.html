    NAME

    Lintian::Index - access to collected data about the upstream (orig) sources

    SYNOPSIS

        use Lintian::Index;

    DESCRIPTION

    Lintian::Processable::Source::Orig::Index provides an interface to collected data about the upstream (orig) sources.
