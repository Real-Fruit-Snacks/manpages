    NAME

    IBUS - dconf database file for IBus

    SYNOPSIS

    /etc/dconf/db/ibus

    DESCRIPTION

    IBus is an Intelligent Input Bus. It is a new input framework for Linux OS. It provides full featured and user friendly input method user interface. It also may help developers to develop input method easily.

    /etc/dconf/db/ibus is a database file dconf and saves the IBus default settings. It can be generated from /etc/dconf/db/ibus.d/00-upstream-settings by dconf update command with a write privilege in /etc/dconf/db. The saved keys and values can be readed by dconf command.

    env DCONF_PROFILE=ibus dconf list /desktop/ibus/
