    NAME

    configure.amr — configuration file for AMR

    DESCRIPTION

    This file is used by amr(1) to autogenerate the Makefile.acr files and/or configure.acr files.

    SYNTAX

    CDIR path1 path2 ;
     
    CSOURCES foo.c bar.c ;
     
    CINCLUDES include1 include2 ;
    Include files used by the src files.
    CDEPS file1 file2;
