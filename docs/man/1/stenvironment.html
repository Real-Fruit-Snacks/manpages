    STENVIRONMENT(1) User Commands STENVIRONMENT(1)

    NAME

    stenvironment - create StepTalk environment process

    SYNOPSIS

    stenvironment [options]

    DESCRIPTION

    stenvironment creates a new StepTalk environment process. The Distributed Objects server is created with name ‘STEnvironment:name’. If no name is given, then ‘UnnamedNN’ will be created. The observer name is a distributed notification observer.

    OPTIONS

    -help
    print this message
    -name name
    set name of environment
    -type env
    use scripting environment description with name env
    -observer name
    observer name will receive notification
