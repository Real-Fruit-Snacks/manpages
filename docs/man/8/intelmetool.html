    NAME

    intelmetool - manual page for intelmetool v1.1 -- Copyright (C) 2015 Damien Zammit

    DESCRIPTION

    usage: debian/coreboot-utils/usr/sbin/intelmetool [-vh?smdb]

    -v | --version:
    print the version
    -h | --help:
    print this help
    -d | --debug:
    enable debug output
    -m | --me
    dump all me information on console
    -b | --bootguard
    dump bootguard state of the platform
