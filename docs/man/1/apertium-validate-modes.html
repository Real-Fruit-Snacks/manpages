    APERTIUM-VALIDATE-MODES(1) General Commands Manual APERTIUM-VALIDATE-MODES(1)

    NAME

    apertium-validate-modes — intermediate modes XML validator for Apertium

    SYNOPSIS

    apertium-validate-modes input_file

    DESCRIPTION

    This is an intermediate tool from Apertium level 2 engine. You should never have to use it independently.

    It is a script that validates a modes.xml file against the apertium structural modes DTD using the xmllint(1) utility.

    OPTIONS

    It has no options.

    FILES

    input_file
    A modes.xml file
    modes.dtd
    The DTD used to validate the input file.
