    NAME

    django_extensions - autoradio player daemon

    SYNOPSIS

    autoplayerd <action> [options] (action=start|stop|restart|run|version)

    DESCRIPTION

    django_extensions is not installed; I do not use it

    autoplayerd daemon for autoradio suite

    OPTIONS

    --version
    show program's version number and exit
    -h, --help
    show this help message and exit
    --pidfile=PIDFILE
    PID filename (default /tmp/autoplayer.lock)
    --stdin=STDIN
    stdin filename (default /dev/null)
    --stdout=STDOUT
    stdout filename (default /tmp/autoplayer.log)
    --stderr=STDERR
    stderr filename (default /tmp/autoplayer.err)
    --user=USER
    user name or id (default none)
    --group=GROUP
    group name or id (default none)
