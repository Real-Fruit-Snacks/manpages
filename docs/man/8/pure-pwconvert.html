    NAME

    pure-pwconvert - Generate a virtual users file from system accounts

    SYNTAX

    pure-pwconvert

    DESCRIPTION

    This program scans system accounts (/etc/passwd) and outputs a FTP virtual users list, suitable to the pure-pw command.

    FILES

    /etc/passwd
    /etc/shadow
    /etc/master.passwd

    AUTHORS

    Frank Denis <j at pureftpd dot org>
