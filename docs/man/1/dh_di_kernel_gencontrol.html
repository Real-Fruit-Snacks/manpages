    NAME

    dh_di_kernel_gencontrol - generate debian/control files for kernel udebs

    SYNOPSIS

    dh_di_kernel_gencontrol [debhelper options]

    DESCRIPTION

    dh_di_kernel_gencontrol is a debhelper program that runs "kernel-wedge gen-control" to generate "debian/control" files for kernel udebs.

    It silently does nothing if the "kernel-versions" file does not exist.

    SEE ALSO

    debhelper(7), kernel-wedge(1)

    This program is a part of dh-di.
