    NAME

    transmission-remote-gtk - RPC client for the Transmission bittorrent client.

    SYNOPSIS

    transmission-remote-gtk [OPTIONS] [torrent file|magnet link|url]

    DESCRIPTION

    transmission-remote-gtk is an application for remote management of the Transmission BitTorrent client using its RPC interface.

    OPTIONS

    The following options are accepted when running "transmission-remote-gtk":

    -m, --minimized, /m
    Start the application minimized
