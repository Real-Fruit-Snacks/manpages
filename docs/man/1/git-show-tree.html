    NAME

    git-show-tree - show branch tree of commit history

    SYNOPSIS

    git-show-tree

    DESCRIPTION

    Show the decorated graph view of one liner summarized commits from all branches.

    EXAMPLES

    Output the commit history log for all branches as tree view:
