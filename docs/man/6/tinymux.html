    NAME

    tinymux - The TinyMUX mush server

    DESCRIPTION

    This manual page documents the tinymux server, a mud server in the MUSH family, only briefly.

    It provides a number of features including support for Unicode, IPv6, SSL, rich color, external module support, and connectivity to external SQL databases. Using its internal programming language, MUSHcode, this server can be used to build rich, virtual, multi-player environments.

    TinyMUX does not provide a superset of all the features available in other MUSH flavors, but those it does implement provide a stable and robust environment that performs as well as or better than that of its peers.
