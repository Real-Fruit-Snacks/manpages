    NAME

    freenect-cppview - manual page for freenect camera test

    SYNOPSIS

    freenect-cppview

    DESCRIPTION

    This is a simple demo program that illustrates the use of the C++ libfreenect library. It also provide a simple way to test the communication with the freenect device.

    Shortcuts

    ESC
    Exit the program
    Key W
    Increase tilt
    Key S
    Reset tilt to 0
    Key X
    Decrease tilt
    Key E
    Set tilt to 10
    Key C
    Set tilt to -10
    Key F
    Change video format
    Key 1
    Change LED color to still green
    Key 2
    Change LED color to still red
    Key 3
    Change LED color to still yellow
    Key 4
    Change LED color to blinking green
    Key 5
    Change LED color to blinking green
    Key 6
    Change LED color to blinking red and yellow
    Key 0
    Switch off the LED
